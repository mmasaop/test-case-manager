# Chromeで記録で「Webサイトのブラウザ自動操作制限の回避を試みる」をOFFにして記録

**関連チケット**: ROBOOPE_1-1836

**重要度**: normal

## テストステップ

### ステップ 1

**操作**: コマンドツールバーから、Chromeで記録 をクリックする

**期待結果**:
Webページの指定 ダイアログが表示される

### ステップ 2

**操作**: 詳細オプションを開く

**期待結果**:
* 「Webサイトのブラウザ自動操作制限の回避を試みる」がOFFであること
* 「Webサイトのブラウザ自動操作制限の回避を試みる」のツールチップが次の文言であること。
    「ロボットによるブラウザの自動操作がWebサイトによって制限されてしまう場合に、このオプションの利用をご検討ください。
    Webサイトの利用規約によっては自動操作が禁止されている場合がありますので、事前に各サイトの利用規約をご確認ください。
    ※このオプションを有効にしても、自動操作が保証されるわけではありません。」

### ステップ 3

**操作**: URLに https://google.com を指定し、記録を開始する

### ステップ 4

**操作**: google検索フォームに、Web自動化記録機能で任意の文字列の入力と、検索を実行するように操作を記録する

**期待結果**:
以下内容のページが表示され、自動操作がブロックされる

「お使いのコンピュータ ネットワークから通常と異なるトラフィックが検出されました。このページは、リクエストがロボットではなく実際のユーザーによって送信されたことを確かめるものです。」
