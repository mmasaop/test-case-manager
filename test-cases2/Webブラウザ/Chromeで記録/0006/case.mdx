# Chromeのユーザーデータ保存フォルダが指定されていて、プロファイル名がDefaultならプロファイル選択画面が発生せずブラウザ起動できる

**関連チケット**: ROBOOPE_1-1916

**優先度**: medium

**重要度**: normal

## テストステップ

### ステップ 1

**操作**: Chromeで記録 を押下し「Webページの指定」ダイアログを開く

### ステップ 2

**操作**: 任意のURLを指定し、ユーザーデータを保存（ログイン状態を保持）をチェックONで記録を開始する

### ステップ 3

**操作**: ブラウザのプロファイルアイコンをクリックし、任意の名前でプロファイルを追加する。
プロファイル追加後、記録用Webブラウザを閉じる。

### ステップ 4

**操作**: コマンドオプションで
・ユーザーデータ保存フォルダ : 指定あり
・プロファイル名 : Default
の状態で、
コマンドテーブルの「Web Chromeを起動」を実行する

**期待結果**:
ブラウザの起動に成功すること
