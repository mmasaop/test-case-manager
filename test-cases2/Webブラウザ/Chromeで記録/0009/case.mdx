# ユーザープロファイルを一覧から選択できる

**関連チケット**: ROBOOPE_1-1925

**重要度**: normal

## テストステップ

### ステップ 1

**操作**: Chromeで二つ以上のプロファイルを作成しておき、ブラウザを閉じる

### ステップ 2

**操作**: ツール→WebからChromeで起動コマンドを追加し有効なURLを入力
%user.home%^AppData\\Local\\Google\\Chrome\\User Data
をユーザーデータ保存フォルダにコピー

### ステップ 3

**操作**: プロファイル名の参照ボタンをクリック

**期待結果**:
設定したプロファイルを選択できるダイアログが表示される

### ステップ 4

**操作**: Default以外のプロファイルを選択する

**期待結果**:
入力欄に値が挿入される

### ステップ 5

**操作**: コマンドを実行する

**期待結果**:
指定したプロファイルでブラウザが起動される

### ステップ 6

**操作**: ブラウザを閉じる
変数を設定コマンドでプロファイル名を変数定義し、
ブラウザの起動コマンドでプロファイル名に変数の値を追記する

### ステップ 7

**操作**: コマンドを実行する

**期待結果**:
指定したプロファイルでブラウザが起動される

### ステップ 8

**操作**: ユーザーフォルダ名に不正な値を入力する
プロファイル名の参照ボタンをクリック

**期待結果**:
エラーが発生しダイアログが表示される
ダイアログの内容がわかりやすく誤字脱字がない
