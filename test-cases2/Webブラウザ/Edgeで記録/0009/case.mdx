# ロボオペを管理者実行した際にEdgeのWeb自動化が正常に起動する事。

**重要度**: normal

## テストステップ

### ステップ 1

**操作**: ロボオペを管理者権限で実行する

**期待結果**:
ロボオペが正常に起動する事

### ステップ 2

**操作**: Edgeで記録コマンドを選択する

**期待結果**:
Webページの指定ダイアログが表示される

### ステップ 3

**操作**: ダイアログのURLにDataに記載しているURLを記載し、記録を開始する

**期待結果**:
全画面でサイト先のページ及び記録ツールが表示されること

**データ**: [https://training.asirrera.com/web/form/](https://training.asirrera.com/web/form/)

### ステップ 4

**操作**: ツールの記録終了を押下する

**期待結果**:
エラーが発生することなく正常にタブ及びツールが終了する事
