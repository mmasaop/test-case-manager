# 「Webサイトのブラウザ自動操作制限の回避を試みる」をOFFにして実行し、ボット検出される

**関連チケット**: ROBOOPE_1-1836

**重要度**: normal

## テストステップ

### ステップ 1

**操作**: 上部メニューのツール > その他コマンド > Webブラウザ > Web Edgeを起動 をクリックしてコマンド追加する

**期待結果**:
Web Edgeを起動コマンドが追加される

### ステップ 2

**操作**: コマンドオプションを確認する

**期待結果**:
「Webサイトのブラウザ自動操作制限の回避を試みる」がOFFであること

### ステップ 3

**操作**: URLに、任意のGoogle検索結果のURLを指定し、コマンド実行を開始する

**期待結果**:
以下メッセージのページに遷移され、Web自動化がブロックされる

「お使いのコンピュータ ネットワークから通常と異なるトラフィックが検出されました。このページは、リクエストがロボットではなく実際のユーザーによって送信されたことを確かめるものです。」
