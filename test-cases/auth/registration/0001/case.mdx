# ユーザー登録機能 - 正常系テスト

![](./test.png)

## テスト概要
新規ユーザーが正常にアカウントを作成できることを確認する

## 前提条件
- メールアドレスが未登録であること
- メール受信環境が準備されていること

## テスト手順

1. **登録ページへアクセス**
   - URLに `https://example.com/register` を入力
   - 登録フォームが表示されることを確認

2. **ユーザー情報の入力**
   
   <FormData>
     name: "山田 太郎"
     email: "yamada.taro@example.com"
     password: "SecurePass123!"
     confirmPassword: "SecurePass123!"
     terms: true
   </FormData>

3. **利用規約の確認**
   - 利用規約リンクをクリック
   - 新しいタブで利用規約が表示されることを確認
   - チェックボックスにチェックを入れる

4. **登録ボタンをクリック**
   - 「アカウントを作成」ボタンをクリック
   - プログレスバーが表示されることを確認

5. **確認メールの受信**
   - 登録したメールアドレスに確認メールが届くことを確認
   - メール内のリンクをクリック

   ![確認メール](./confirmation-email.png)

6. **アカウントの有効化**
   - 確認リンクをクリック後、成功メッセージが表示される
   - 自動的にログイン画面へ遷移する

## 期待される結果
- アカウントが正常に作成される
- 確認メールが5分以内に届く
- データベースにユーザー情報が正しく保存される
- パスワードはハッシュ化されて保存される

## 検証項目
- [ ] フォームバリデーションが動作する
- [ ] パスワード強度インジケータが表示される
- [ ] 重複メールアドレスのチェックが機能する
- [ ] XSS対策が実装されている

- 確認メールの有効期限は24時間
