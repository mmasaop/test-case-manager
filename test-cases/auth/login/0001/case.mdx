# ログイン機能 - 正常系テスト

## テスト概要
通常のユーザーアカウントでログインできることを確認する

## 前提条件
- テスト用アカウントが作成済みであること
  - Email: test@example.com
  - Password: Test123!
- ブラウザのキャッシュがクリアされていること

## テスト手順

1. **ログインページへアクセス**
   - URLに `https://example.com/login` を入力
   - ログインページが表示されることを確認

   ![ログイン画面](./login-page.png)

2. **認証情報の入力**
   ```
   Email: test@example.com
   Password: Test123!
   ```

3. **ログインボタンをクリック**
   - 「ログイン」ボタンをクリック
   - ローディングインジケータが表示されることを確認

4. **ダッシュボードへの遷移を確認**
   - ダッシュボードページへリダイレクトされること
   - ユーザー名が画面右上に表示されること

## 期待される結果
- ログインが成功し、ダッシュボードが表示される
- セッションCookieが正しく設定される
- ユーザー情報が画面に表示される
