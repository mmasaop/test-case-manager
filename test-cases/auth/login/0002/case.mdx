# ログイン機能 - 異常系テスト（パスワード誤り）

![](./vite.svg)

## テスト概要
誤ったパスワードでログインを試みた際に、適切なエラーメッセージが表示されることを確認する

## 前提条件
- テスト用アカウントが作成済みであること
  - Email: test@example.com
  - Password: Test123! （正しいパスワード）
- ブラウザのキャッシュがクリアされていること

## テスト手順

1. **ログインページへアクセス**
   - URLに `https://example.com/login` を入力
   - ログインページが表示されることを確認

2. **誤った認証情報の入力**
   ```
   Email: test@example.com
   Password: WrongPassword123
   ```

3. **ログインボタンをクリック**
   - 「ログイン」ボタンをクリック

4. **エラーメッセージの確認**
   - エラーメッセージが表示されること
   - メッセージ内容: "メールアドレスまたはパスワードが正しくありません"

   ![エラーメッセージ](./error-message.png)

## 期待される結果
- ログインが失敗する
- エラーメッセージが赤色で表示される
- 入力したメールアドレスは保持される
- パスワードフィールドはクリアされる
- ログイン試行回数がカウントされる

## セキュリティ要件
- 具体的にどちらが間違っているかは表示しない
- 5回連続で失敗した場合、アカウントが一時的にロックされる
- ロック時間: 15分間

## 備考
- エラーメッセージはユーザーフレンドリーであること
- SQLインジェクション対策が実装されていることを確認
